### Week 3 Objectives:

* Learn about AWS compute and storage services: EC2 (VM), Lightsail, EFS/FSx.
* Explore AWS Application Migration Service (MGN).
* Deploy AWS Backup and practice creating backup plans, setting up notifications, and testing restores.
* Get familiar with AWS Storage Gateway, practice creating file shares, and mount them on an on-premises machine.
* Start with Amazon S3: create a bucket, enable static website hosting, configure public access and object permissions, test the website, and accelerate it with CloudFront.
* Conduct architectural research for the AWS team project.
