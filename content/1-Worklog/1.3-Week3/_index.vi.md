### Mục tiêu Tuần 3:

* Tìm hiểu về các dịch vụ compute và storage của AWS: EC2 (VM), Lightsail, EFS/FSx.  
* Khám phá AWS Application Migration Service (MGN).  
* Triển khai AWS Backup và thực hành tạo kế hoạch sao lưu, thiết lập thông báo và kiểm tra khôi phục.  
* Làm quen với AWS Storage Gateway, thực hành tạo file shares và gắn kết chúng vào máy on-premises.  
* Bắt đầu với Amazon S3: tạo bucket, bật tính năng static website hosting, cấu hình quyền truy cập công khai và quyền đối tượng, kiểm tra website, và tăng tốc bằng CloudFront.  
* Thực hiện nghiên cứu kiến trúc cho dự án nhóm AWS.

### Kết quả đạt được tuần 3:

* Học và thực hành các dịch vụ AWS sau:

  * **Compute:** EC2 (VM), Amazon Lightsail  
  * **Storage:** Amazon EFS, Amazon FSx, Amazon S3, AWS Storage Gateway  
  * **Migration:** AWS Application Migration Service (MGN)  
  * **Backup:** AWS Backup (tạo kế hoạch sao lưu, thiết lập thông báo, kiểm tra khôi phục)

* Triển khai thành công AWS Backup: tạo backup plan, thiết lập thông báo và kiểm tra khôi phục.  

* Tạo và gắn kết File Storage Gateway vào máy on-premises.  

* Cấu hình Amazon S3 static website hosting và tăng tốc bằng CloudFront.  

* Hoàn thành nghiên cứu kiến trúc ban đầu cho dự án nhóm AWS.
